This is an web app built for tracking playstation games. It allows users
to each log in by inputing their first and lastname, email address,
set a password and also upload a profie picture. Once logged in,
users can and create their own collections with a name and image.
Inside collections users can create a game object with a name,
platform, trophy count of the game, and image of the game. Once created games
can be deleted and edited the same as collections. There are statistics visible
to the user, on the homepage featuring individual stats and on the about page
global app statistics the same as on the signup page. This is a complete
and (mostly) working version of the app that does what i intended it to do.
I hope you enjoy using this app.
